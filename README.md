# Matheus Henrique | Portfolio

> Portfólio pessoal com estética de terminal VS Code e chuva de código Matrix.

## ✨ Features

- 🖥️ Estética de Terminal / VS Code
- 🌧️ Chuva de código Matrix em TypeScript/Node.js real
- 🌓 Tema claro e escuro com persistência via localStorage
- 📱 Totalmente responsivo (mobile-first)
- ⚡ Zero dependências — HTML, CSS e JavaScript puro
- 🎯 Scroll reveal animado nas seções
- 🔗 Links ativos no header ao navegar

## 📁 Estrutura

```
portfolio/
├── index.html          # Página principal
├── css/
│   └── style.css       # Estilos completos (variáveis de tema)
├── js/
│   └── main.js         # Matrix rain, tema, scroll, animações
├── assets/
│   └── curriculo.pdf   # Adicione seu currículo aqui
└── README.md
```

## 🚀 Deploy no GitHub Pages

1. Faça upload de todos os arquivos no repositório `Portifolio`
2. Vá em **Settings → Pages**
3. Em *Source*, selecione **Branch: main** → pasta **/ (root)**
4. Clique em **Save**
5. Acesse: `https://Mhenrique115.github.io/Portifolio`

## ✏️ Personalização

Edite o `index.html` para atualizar:

- **Seu nome e bio** — Seção `#sobre`
- **Links do GitHub/LinkedIn/Instagram/E-mail** — Seção `#contato`
- **Projetos** — Seção `#projetos` (substitua os `href="#"` pelos links reais)
- **Conquistas** — Seção `#conquistas`
- **Currículo** — Coloque o PDF em `assets/curriculo.pdf`
- **Foto** — Substitua o bloco `.profile-frame` por uma tag `<img>`
